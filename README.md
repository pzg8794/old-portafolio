# Old Portafolio Hub

Canonical old Drive folder: https://drive.google.com/drive/folders/1YuT0Zl11GJgz0nGuqTAxvNCrWmDK6F9d

This repo is the lightweight navigation hub for the old pre-2024 Portafolio. The useful project archive areas are top-level; individual project repositories are linked from their sections as they are created from Drive.

## Main Areas

- [Projects-Code](./Projects-Code/README.md) - Source-code and software project work from the old Portafolio Drive folder. Individual project repositories should be linked here as submodules or direct repo links.
- [Projects-No-Code](./Projects-No-Code/README.md) - Non-code coursework and academic project evidence such as reports, videos, math/physics/course artifacts, and writing projects.
- [Publications](./Publications/README.md) - Old reports, papers, and publication-style portfolio outputs.
- [Resume-CV](./Resume-CV/README.md) - Resume, CV, interview prep, candidate profile, references, cover letters, and old front-facing portfolio documents.
- [Persona](./Persona/README.md) - Personal or identity-adjacent legacy material. This area must stay private and should be reviewed carefully before sharing.
- [Work-Projects](./Work-Projects/README.md) - Client/employer/project work. Keep private unless each item is reviewed for permission and sensitive data.
- [PHD](./PHD/README.md) - Older graduate-school/program research material kept as historical reference only.

## Organization Rule

Keep this repo light. Put real project content in the individual GitHub repos and link them here as submodules or repo links. Keep personal and work/client material private unless explicitly reviewed.
