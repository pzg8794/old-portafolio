# Old Portafolio Index

Canonical Drive folder: https://drive.google.com/drive/folders/1YuT0Zl11GJgz0nGuqTAxvNCrWmDK6F9d

This repository indexes the old pre-2024 `Portafolio` folder. The heavy project archive remains in Google Drive; git is used here for navigation, notes, and selected small source files.

## Top-Level Folders

- [Learning and Job Prep](./Learning%20and%20Job%20Prep/README.md) - Use this for older interview prep, job-search records, coding practice, and learning reference material.
- [Legacy DocPortfolio Code](./Legacy%20DocPortfolio%20Code/README.md) - Use this for the old DocPortfolio source file/code artifact.
- [Portfolio Documents](./Portfolio%20Documents/README.md) - Use this first for old resume, cover letter, portfolio layout, work references, and presentation/source portfolio documents.
- [Projects Archive](./Projects%20Archive/README.md) - Use this for pre-2024 technical, academic, work, and publication project evidence. Keep bulky project files in Drive; this repo indexes them for navigation.

## Repository Files

- [.gitignore](./.gitignore) - Keeps local Finder/editor noise out of the portfolio index.
- [OLD_PORTAFOLIO_SOURCE.md](./OLD_PORTAFOLIO_SOURCE.md) - Records the canonical Drive folder, cleanup scope, and why this repo is limited to the old pre-2024 portfolio.
- [README.md](./README.md) - This front-door index for the old portfolio repo.

## How To Use This Repo

- Start with `Portfolio Documents` for the old portfolio/front-facing materials.
- Use `Projects Archive/Publications`, `Projects Archive/Projects Code`, and `Projects Archive/Work Projects` to find evidence for applications.
- Use the Drive link for bulky files, old binaries, media, and complete project folders.
- Every local directory has a `README.md` listing the immediate Drive items it contains or references.
