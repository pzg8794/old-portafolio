# Projects Archive

Drive location: `Projects Archive` inside the old Portafolio folder.

Purpose: Use this for pre-2024 technical, academic, work, and publication project evidence. Keep bulky project files in Drive; this repo indexes them for navigation.

## Items

- `PHD` (folder) - Old graduate-school/program research material kept as historical reference.
- `Projects Code` (folder) - Old source-code and coursework project folders, organized by language/course/project area.
- `Projects(No Code)` (folder) - Old non-code coursework/project evidence such as reports, presentations, media, and class assignments.
- `Publications` (folder) - Old publication/report outputs and project papers that may be useful as application evidence.
- `Resume` (folder) - Old resume and interview preparation files; reference for historical career materials only.
- `Stuff` (folder) - Miscellaneous old documents and archives; review before using in applications.
- `Work Projects` (folder) - Old work/client/project artifacts. Review permissions/sensitivity before sharing externally.
- `Please assess the extent to which your work experience provided you with the opportunity to accomplish the items below.docx` (file) - Text/source document; review for old portfolio language or application evidence.
- `texture.jpg` (file) - Image/media asset; inspect before reusing in a current application packet.
